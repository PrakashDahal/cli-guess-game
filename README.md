# Guess the number

|Levels     |Number|
--- | --- | 
|Easy       |10    | 
|Medium     |50    | 
|Hard       |100   | 
|SuperHard  |1000  | 

- A command line game.
- Max attempt is 3.
- Guess the number in 3 attempts.

```
npx cli-guess-game
```

*Note: You must have node and npx version greater than 12 in your machine to run this.*
